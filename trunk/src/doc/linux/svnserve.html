= svnserve = Below a simple svnserve setup with following steps * create an svn user * create repository as /home/svn/svn using svnadmin * edit config file svn/conf/svnserve.conf * edit password file svn/conf/passwd * ensure that svn restarted via the cron on @reboot 
# adduser svn
# su - svn
$ svnadmin create /home/svn/svn # create a repo "svn" in svn home dir

$ ls svn/conf/
authz
passwd
svnserve.conf

svn@evans:~$ vi svn/conf/svnserve.conf

[general]
anon-access = read
auth-access = write
password-db = passwd

svn@evans:~$ vi svn/conf/passwd
[users]
evans = evansSVNpasswd

$ crontab -e
@reboot svnserve -d --listen-host evans -r /home/svn/svn
where the --listen-host option is on the external IP interface rather than localhost. [[Category:uncategorized]] __NOTOC__
