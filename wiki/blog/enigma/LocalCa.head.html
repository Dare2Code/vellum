We consider a bunch of remote clients connecting to our private Java server using SSL sockets with client authentication. Naturally, we generate a private key for each client, and perhaps import all our clients' self-signed certificates into our server truststore. In this article, we'll sign the client certificates using a local CA certificate. In this case our truststore contains our CA certificate only. However, our server should somehow support certificate revocation so that we can disable access e.g. for devices that go amiss.
