If it's gonna take a while to start up our Swing app, then we probably wanna display a splash screen. It's easy with Java6 - just add a ~-splash command-line option.

        if (splash == null) {
            throw new Exception("no splash image specified eg. -splash:mysplash.png");
        }
        splashBounds = splash.getBounds();
        splashGraphics = (Graphics2D) splash.createGraphics();
